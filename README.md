- 2020/9/1
------------
在民道群里跟一些人就token的价值进行了一些辩论，有人点赞fork Uniswap的sushi，感慨dex没有护城河，只要设计好代币经济模型，fork一下就能轻而易举取代原来的项目。对此本人不以为然，token的价值是由项目决定的，而不是token决定项目的价值，这是我的核心论点。至于论据，一方面正如民道所说，如果单通过token模型的创新就可以吸引用户，那么fork之后的项目也会面临其他人的再次fork，依次类推，成了一个击鼓传花的游戏；最主要的则是因为真正吸引用户的一定是项目自身的价值，靠token吸引用户充其量算是短期促销活动，但是用户的长期黏性靠的是产品本身。很显然，Uniswap的团队浸淫dex多年，经验成熟，路线图清晰。而sushi，无论其背后的团队还是其开发路线图，都不清晰。群氓一时的热情能持续多久？让我们拭目以待。

CRV暴涨22%，收报4.5刀。虽然预料到昨天的新治理投票是一个利好，但是没想到市场反应这么激烈。由此也可见，牛市确实已来，稍微有些利好币价就会被拉升。然而，最近LINK币价的表现让我很困惑。客观的评级，LINK的基本面并没有发生多大的变化，要么是之前涨太猛，现在正处在回调阶段；要么就是为了蓄力，一举突破之前的新高。我更倾向于后一种判断，永远不要低估LINK狗庄的力量。有人曾把LINK比作XRP，要么不涨，一涨惊人，所以我会考虑继续增加LINK3倍杠杆头寸，期待其亮眼的表现。ETH币价的表现颇为抢眼，正在逐渐恢复到今年新高的位置，这对于上周才跌回380刀的ETH来说无疑打了一剂强心针。AMPL已经回本，对于这个币我既没有加仓也没有减仓，28刀的成本总价，预期涨到100刀即出货。话说回来，国内三大交易所似乎还没上AMPL？PNK最近一段时间币价表现并不抢眼，甚至可以称之为四平八稳。考虑到即将上架火币，希望能吸引一波资金，再掀起一轮小高潮。MKR仍然坚定看好，不被短期币价干扰，继续长持策略。LRC是国内为数不多踏实做事的项目，其在二层扩容的探索值得钦佩，1万个LRC计划拿到2刀，冲冲冲！AAVE昨天赞助EPNS的新闻让我慢慢开始欣赏这个项目，持有的3000个LEND短期没有出售的打算。AAVE让我印象比较深的是团队成员积极参与、解答社区问题，鼓励社区治理，可以说属于草根团队努力拼搏的典范。所以我决定抛弃意识形态，向一切努力、敬业的团队致意。距离去年SNX质押解锁的日期越来越近，只希望到时候市场还没崩盘，最好SNX能涨到10刀，嘻嘻嘻。继续持有TRB，预言机赛道的热度仍然不会停歇。提前布局MCB和RPL，一个是国产的去中心化衍生品交易所，一个是以太坊2.0基础设施服务商。不过昨天看了下MCDEX的AMM交易数据，24小时只有可怜的几万刀成交量，手续费几百刀，好可怜！

- 2020/9/2
--------------
Time flies. 转眼记日记已经三天，昨晚在币乎主持了一场去中心化合约的直播，邀请到MCDEX的刘杰和Perptual的彦文。整场直播干货多多，当大家询问以太坊成为DeFi的阵地原因除了先发优势和网络效应还有其他什么原因的时候，刘杰的那句资产是以太坊最大的护城河让我印象深刻。现在想想，以太坊上的ETH稳居第二的市值，又通过WBTC吸收一部分BTC到以太坊上来，再加上各类ERC20代币，尤其是DeFi届的高质量代币如LINK、MKR、DAI等，优质资产实在太多。资产、可组合性、网络效应、先发优势，今天出地铁我又想到曾经我对以太坊成功的分析文章，那就是人才！以太坊上有太多的聪明人了，真的是太多太多。国内人只刷微博，怎么可能了解到以太坊海外社区的强大呢？

昨天下午有点FOMO了，以为DODO会是下一个YFI，慌着把币安上的BNB换成SNX要朝DODO流动池里充值，结果才意识到提币需要谷歌二次验证码，而我的谷歌验证码在另外一部手机上没带到公司来。当时那个急啊，赶紧去OKEX上卖了一部分MKR，再换成SNX提现。结果提了半小时收到提现成功短信，却发现metamask没收到币。查了下提币交易详情，状态失败，当时心有点凉，以为币要丢了。赶紧去Synthetix的discord群求助，结果才意识到上午Synthetix就发升级公告，合约部署期间SNX转账功能无法使用，操。。其实上午就看到这个通知，但是下午真的是迫不及待的想充值啊，直接忽略这个公告了。由此可见，FOMO太害人了，竟然让自己变得如此不理性。话说回来，为什么要换成SNX冲到DODO中呢，很简单因为DODO这次的流动性挖矿工支持5个交易对，其中一个是SNX/USDC交易对，而每个交易对的两个币分别代表两个池子，也就是起初一共10个池子。这10个池子每个区块各奖励同等数额的DODO代币，这也就意味着你在哪个池子里占比多，你获得的代币越多。当时我点开了每个交易对，发现只有SNX和COMP相对占比小，充值235个SNX就可以占到整个池子千分之一的比例。我下午的时候还纳闷为什么SNX池子中的代币数量一直没增加，后来结合Synthetix公告才明白这特么不是大家不愿意充值而是正好赶上升级冲不了啊。现在想想，这种可组合性对各个项目的影响还是很大的，试想一下假如某个场景下用户急需对SNX进行操作而又因为Synthetix升级操作不了，画面太美不敢想。我真的挺担心聪明的黑客可能会利用这一点搞些事情，当然，这种升级期间影响所有的链上活动，所以真的搞事情估计也是在cex内搞。

昨天把剩下的YFII在7000刀左右卖了，虽然今天一早醒来看到到了9000刀。但是我必须要意识到，这本就不该是属于我的财富，所以不存在错过，更不是认识不到位的问题，非要找个理由，就是不合适吧。YFII不是价值投资标的，我也看不到其未来，这就是我的基本观点。虽然大家都在聊SUSHI，我觉得这玩意儿也挺扯淡的，尤其是看到曹寅在朋友圈疯狂推销这俩项目，我已经屏蔽他的朋友圈了，很担心他成为下一个薛蛮子/李笑来，但看来是不可避免的了。昨天还在460刀的时候卖了1个ETH，然后今天到了480刀，当然有点小后悔，但毕竟还是赚到了，不是吗？也把自己持有的ETHUP杠杆token卖了，拿了大半个月，赚了10%，安慰自己总比亏好。昨晚在币安上是用BNB买的SNX，结果今天BNB暴涨10%，SNX相对来说暴跌，一番操作，不如不操作，我累个操。。。。感觉现在的DeFi圈只适合观察，不适合参与，或者这么说不适合小户参与。昨天跟少平聊天，他说他也在玩流动性挖矿，但是人家用的一小部分资金也许就是我的全部身家，如果我一直对标这些人那么我只会更痛苦和迷茫。因此，于我而言，此时此刻的坚守更能体现我是否真的遵从价值投资导向。昨天发生了太多，ETH的GAS费创历史新高，达到了均价450GWEI的咋舌程度。

- 2020/9/3
---------------
哎，又跟人吵架了，这次是在潘超的Maker群。感觉自己太刚了，现在想想好没必要。因为觉得Maker的量化宽松政策也是这次DeFi狂热的推手，于是就感慨是罪魁祸首之一，哈哈确实言论显得耸人听闻。结果引起了一众人的反驳，意思是就DAI这点体量，还不至于引发太大的风险。其实我也挺情愿自己的担心是多余的，但是呢在金融行业的履历以及阅读各种股市牛市群氓疯狂的书籍让我觉得这次DeFi必然以极其惨烈的方式收场，虽然我不知道什么时候会发生，但一定会发生，所以我看空不做空，而且会慢慢套现。删了林明的微信好友，感觉道不同不相为谋，还记得去年聊Synthetix的时候他主动加我后研究Synthetix白皮书后的大肆吹捧，后来SNX币价下跌就很少看到他谈这个项目了。林明研究项目的积极性很高，但是抱有太强的功利性目的，投机性强。我对这种项目方公布稍微一些进展就鼓吹重大创新的人十分鄙视，说实话比我还浮夸，还随便。最近他又开始奶YFI，yearn当然有价值，但也没有那么夸张的价值，而且yearn很有可能像长期资本管理公司一样因为对其套利算法过于自信而导致致命失误，从而引发系统性危机。同样，看空不做空，不参与也不羡慕。

Thorchain的bepswap终于与大家见面了，链闻报道了这个新闻然而国内讨论的用户并不多，再一次为国内用户错失一个价值项目感到悲哀。因为Thorchain的中文电报群，我发现一个很有意思的现象，很多优质项目的中文电报群都是台湾老乡们在维护，比如Thorchain，Curve，乃至Dhedge。包括前段时间因为Kleros结识的一位PNK持有者，也是台湾人。大陆同胞们真的要向台湾老铁们学习啊，看看他们在电报群里聊项目，虽然也会谈谈币价，但是很显然对项目有深刻的认知，他们持有是建立在对项目的信心基础上，而大陆太多玩家纯粹是买了待涨，不管不问项目自身的价值。话再扯回来，bepswap就是Thorchain基于币安链推出的一个去中心化交易所，当然未来Thorchain会迁移到自己的链上，而bepswap也将连接不同链上的资产。项目方介绍bepswap时就说我们是多链版Uniswap，我觉得这种解释形象而又省事。不过刚退出，可质押的RUNE有上限约束：15万枚RUNE，据说8分钟内就质押满了。但是令人无语的是，一个理性的流动性提供者应该提供等值的RUNE和另外一种资产如BNB，而实际上是很多RUNE持有者不清楚流动性提供者的角色，还以为这里的质押与在runevault上相似，抵押RUNE就完事了，结果导致池子里的BNB相对价值提升，一下子把BNB价格推高到38刀，然后被一些聪明的用户手动搬砖套利把价差抹平了。另外我个人觉得Thorchain团队使用stake一词确实容易引起歧义，因为stake很容易让用户把这种操作与runevault的stake等价起来，而实际上这里的stake本质上是liqudity provider，充当流动性提供者的角色，当时我建议使用deposit，或者再简单点当用户提供流动性的时候，必须提供两类资产，以免小白用户在未清楚bepswap运行机制的情况下误操作而造成资产损失。但是这也引发了另外一个问题，即在用户操作前，应该清楚自己在做什么，而不是指望项目方帮忙解答一切问题或者自认为该应用是这样或那样运行的。由此也可以看出，FOMO情绪太吓人，所有人盯着质押RUNE带来的高收益，却从没人想过其中隐藏的一些风险。结果就是看到很多用户说不清楚自己质押的RUNE数量为什么一直在减少，另外想取出来操作不了，这就是典型的先操作再研究而不是正确的先研究再操作。七哥庆幸自己的后知后觉，没有急着操作，但是呢也感慨自己对bepswap亦一知半解，需要补充相关知识。

晚上在民道群看到币安智能链正式上线，集合了一些国产DeFi项目共建生态，结果如何，且走且看。不过呢，现在钱包还没上，合作的DeFi项目也没部署，一切都还不确定，我并不着急入场，但是呢会保持关注。另外吐槽一下他们写的中文白皮书，更像是一个说明书，排版粗糙，偶现错别字，待改进的地方还很多。

受到一些不明利空因素的影响，在币安挂单的单价20刀的LINK3倍杠杆代币成交，在OKEX挂单的PNK也以0.13刀的单价成交，最近操作实在是有些频繁啊，不知道给交易所贡献多少手续费了。

决定聚焦项目，别再被各类项目分散注意力。现在暂定聚焦的项目有：Kleros,Thorchain,AAVE,ZAPPER.FI,CURVE。

再提一嘴，yearn的yweth也上线了，其实就是帮助用户更方便的使用curve，策略是：用户在yweth池存入ETH，该池子自动把ETH质押到Maker的MCD借DAI，再把DAI存入Curve挖矿和赚取LP收益。

- 2020/9/4
-----------------
时光荏苒，9.4已三周年，很遗憾当年没有切身参与到那场恐慌带来的崩盘以及人人开始怀疑币圈前途的时代。不过呢，自从入圈以来，也见证了大大小小的崩盘，这不今天再次上演瀑布行。单ETH就从480刀一路下滑到现在的370刀，暴跌原因未知。但是与此同时，美股也迎来闪崩，让人不仅怀疑两者是否存在某种联系。Anyway，我的LINK3倍杠杆代币损失惨重，昨天挂盘的15刀的价格无情成交，而在前一天的接盘价还是17.95刀，今天打算挂盘11刀，继续接盘。此外，昨天3刀挂盘的CRV也无情成交，而相比昨天的跌幅竟然是30%，着实恐怖呀。市场行情的这种波动让我不禁困惑：在牛市，挂一个极低/极高的价格竟然都有可能成交。不管怎样，对后市依然乐观，毕竟咱也是见惯暴跌大场面的男人。

昨天忙于工作，项目和新闻都没关注太多，推特上一票的被yearn的yweth池刷屏，甚至Maker的创始人RUNE也忍不住发推致谢yearn，帮Maker快速实现5亿DAI的小目标。然而，此举也引发社区一些成员的担忧，担心The DAO重演，而yearn就是下一个The DAO。私下说，我还是挺期待再来一次The DAO的，因为不来一次大家都意识不到风险的重要性，不摧毁当前的泡沫就无法迎来更大的泡沫。哈哈，开玩笑，只是从人类的进程来看，犯错不可避免，覆辙总在重复，因此担心也没太大用。现在的情况是，自己意识到风险明哲保身即可，出来呼吁肯定会被舆论淹没。我以为，现在yearn已经成为defi最大的单点故障隐患，不知道多少黑客正在潜心研究如何盗走里面的代币。至于有人担心yearn离开了Andre之后yearn将何去何从，我以为大家对他过于拔高了，只要一个项目有利可图，总有聪明的人会接手，根本无需担心。

今早看到安仔在群里提到社区分叉了curve，要再造一个代币分发公平的curve。我对于各类项目的fork版，仍然采取观察但不急着参与的策略。通过今天跟群友的讨论，我认为达成了以下共识：既然项目是开源的，就要接受fork版，而不是无端指责或者有任何偏见，这是基调；fork促使项目竞争，无论大家觉得这是良性还是恶性，竞争迫使各个项目方不断创新，赢民心者赢天下；dapp的fork跟公链的差异在于技术门槛相对较低，几乎不存在用户迁移成本，但是我觉得未来dapp可能越来越复杂，用户粘性越来越大，fork很难持续。

今早去OKEX看了下，自己的SNX回来了，心里的一块石头总算落下，以后要谨慎操作呀！

- 2020/9/5

今天看了看Curve的常见问答，也算对Curve有个进一步的认识。Curve是一个去中心化稳定币交易所，其实不太准确，因为它也支持ERC20类BTC的交易对。Curve的不同池子收益策略不同，但是共同点是LP都能获得交易手续费。而cpool除了手续费，还能获得Compound的存款收益。ypool除了手续费，还能获得yearn利率聚合的收益。而有些池子可能有利益相关项目方的代币补贴，如wBTC/sBTC/renBTC池子，除了获得交易手续费，还能获得Synthetix项目方和Ren项目方的代币激励。后来Curve自己发了个治理兼功能型代币CRV，流动性提供者除了获取以上收益，还能获得CRV奖励，也称为流动性挖矿。而LP基本上没啥损失，不过假如只存入一种稳定币，Curve会自动将它按比例兑换成稳定币池的不同种类稳定币，然后发给你一个lp token，也可以认为是所有权证明。而veCRV，全称voting escrow CRV是锁仓CRV获得的投票代币，用于Curve项目治理。锁仓越久，1个CRV越能换更多veCRV。不知道有没有人为了省手续费去Uniswap直接购买veCRV,哈哈。veCRV除了具有投票功能外，还有加速功能。本来LP们提供流动性就可以获得CRV奖励，但是如果veCRV锁仓时间久，则可以获得加加速，即收益的CRV比不锁仓的多。
